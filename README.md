# AI Prompt Studio

## 项目简介

AI Prompt Studio 是一款集成多种 AI 辅助创作、提示词管理、系列流程、统计分析、批量激活等功能于一体的专业工具，支持多主题、多语言、深浅色切换，适用于提示词工程、内容创作、AI 应用开发等多种场景。

## 主要功能

- **提示词管理**：支持提示词的创建、编辑、删除、分类、标签、收藏、置顶、历史记录、批量操作、导入导出等。
- **AI 辅助创作**：
  - 提示词生成器：根据主题和描述自动生成高质量提示词
  - 优化助手：智能分析并优化现有提示词
  - 变体生成、智能补全、结构分析、领域转换、风格适配、语言转换等多种 AI 工具
  - 支持多模型和 API 配置
- **系列管理**：将提示词组织为多步骤系列，支持分组、排序、批量管理、文件夹、搜索等。
- **回收站**：支持提示词的软删除、恢复、永久删除、批量操作、自动清理。
- **统计分析**：可视化展示提示词、分类、标签、收藏、置顶等多维度统计数据。
- **设置中心**：主题切换（深/浅色）、多语言、API 配置、界面行为、历史记录、缓存清理、工厂重置等。
- **帮助中心**：内置使用指南、常见问题、关于信息，支持多语言和主题适配。
- **批量激活/授权工具**：支持机器码生成、激活码生成、批量处理、导入导出等。
- **云存储同步**：支持多云服务商（腾讯云COS、阿里云OSS、七牛云存储），可在不同设备间同步提示词数据库和媒体文件（图片、视频），用户可自由选择云存储服务商。
- **数据库与数据管理**：内置数据库，支持数据优化、自动保存、恢复、导入导出等。
- **插件与脚本**：支持部分自动化和修复脚本，便于维护和扩展。

## 功能特性

### 自动隐藏功能

*版本: 1.0.0*  
*更新日期: 2025年5月31日*

#### 功能概述

自动隐藏功能允许窗口在靠近屏幕边缘时自动隐藏，只留下窗口边缘可见，当鼠标移动到该区域时窗口会自动显示。这个功能类似于QQ等聊天软件的"贴边隐藏"功能，可以帮助用户在不需要时节省屏幕空间，同时保持应用程序随时可用。

#### 使用方法

##### 启用自动隐藏功能

1. 打开设置页面（点击导航栏中的设置图标）
2. 在常规设置中，勾选"启用自动隐藏功能"选项
3. 设置保存后，功能立即生效

##### 日常使用

- **隐藏窗口**: 将窗口拖动到屏幕边缘（上/下/左/右），保持800毫秒后窗口会自动隐藏
- **显示窗口**: 将鼠标移动到窗口隐藏位置的边缘，窗口会自动显示
- **禁用功能**: 随时可以通过设置页面取消勾选"启用自动隐藏功能"来禁用此功能

##### 快捷操作

| 操作 | 描述 |
|------|------|
| 将窗口拖到屏幕边缘 | 触发自动隐藏（需等待800ms） |
| 鼠标移动到隐藏区域 | 立即显示窗口 |
| 双击隐藏窗口的可见部分 | 立即显示窗口 |
| 在设置中切换自动隐藏选项 | 启用/禁用整个功能 |

#### 常见问题

**Q: 为什么我的窗口不会自动隐藏？**  
A: 请确认在设置页面中已启用"自动隐藏功能"选项，并且窗口确实靠近了屏幕边缘。默认设置需要窗口在屏幕边缘停留800毫秒才会触发隐藏。

**Q: 如何防止窗口意外隐藏？**  
A: 如果不希望窗口隐藏，可以在设置页面中禁用"自动隐藏功能"选项。或者避免将窗口停留在屏幕边缘太久。

**Q: 窗口隐藏后如何找回？**  
A: 将鼠标移动到窗口隐藏的屏幕边缘区域，窗口会自动显示。如果找不到确切位置，可以尝试沿着屏幕四边移动鼠标。

**Q: 能否只在某个方向上启用自动隐藏？**  
A: 当前版本不支持指定方向隐藏，此功能在计划中，将在未来版本中实现。

### 云存储媒体文件支持

*版本: 1.0.1*  
*更新日期: 2025年6月1日*

#### 功能概述

云存储功能现已支持提示词中的图片和视频文件同步。用户可以选择性地将媒体文件与数据库一起上传到云存储，实现在不同设备间的完整数据同步。支持腾讯云COS、阿里云OSS、七牛云存储等多个云服务商，用户可根据需求自由选择。

#### 主要特性

- **选择性上传**：用户可以选择是否包含媒体文件进行上传
- **智能打包**：自动将数据库和媒体文件打包成压缩包，优化传输效率
- **完整同步**：支持图片、视频文件的完整路径结构保持
- **向后兼容**：完全兼容仅包含数据库的旧版本文件
- **灵活下载**：下载时可选择是否提取媒体文件

#### 使用方法

##### 配置云存储

1. 打开设置页面，切换到"云存储"标签页
2. 点击"配置云存储"按钮
3. 填入腾讯云COS的配置信息：
   - SecretId：腾讯云访问密钥ID
   - SecretKey：腾讯云访问密钥
   - 地域：存储桶所在地域
   - 存储桶名称：COS存储桶名称

##### 上传数据

1. 在云存储设置页面点击"上传数据库"按钮
2. 在弹出的对话框中选择是否"同时上传图片和视频文件"
3. 点击"上传"开始上传过程
4. 上传完成后会显示成功提示

##### 下载数据

1. 在云存储设置页面点击"下载数据库"按钮
2. 选择要下载的云端文件
3. 选择是否"与本地数据库合并"
4. 选择是否"提取图片和视频文件"
5. 点击"确定"开始下载过程

#### 技术实现

- **压缩算法**：使用ZIP_DEFLATED压缩算法，减少存储空间和传输时间
- **路径保持**：保持媒体文件的相对路径结构，确保引用关系不变
- **错误处理**：完善的异常处理机制，提供清晰的错误信息
- **进度反馈**：实时显示上传/下载进度，避免界面冻结

#### 注意事项

- 包含媒体文件的上传会增加传输时间和云存储空间消耗
- 媒体文件存储在本地`data/media/`目录下
- 数据库中仅存储媒体文件的相对路径引用
- 建议定期备份重要的媒体文件

#### 常见问题

**Q: 为什么我的媒体文件没有同步？**  
A: 请确认在上传时勾选了"同时上传图片和视频文件"选项，下载时勾选了"提取图片和视频文件"选项。

**Q: 旧版本上传的文件还能下载吗？**  
A: 完全兼容，旧版本仅包含数据库的文件可以正常下载，只是不会包含媒体文件。

**Q: 如何节省云存储空间？**  
A: 如果不需要在多设备间同步媒体文件，可以在上传时取消勾选媒体文件选项，仅同步数据库内容。
